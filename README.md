# Jamila's Bakery Delights - Bespoke Showstopper Cakes

🎂 A stunning, fully responsive website for Jamila's bespoke cake business in Leicester.

## Features

✨ **Elegant Design**
- Premium color palette (cream, soft pink, dark berry, and gold)
- Luxury typography with Cormorant Garamond and Montserrat fonts
- Smooth scroll animations with AOS.js

🎨 **Sections**
- **Announcement Bar** - Fast-turnaround availability notice with direct contact link
- **Navigation** - Sticky menu with smooth scrolling
- **Hero Section** - Eye-catching headline with call-to-action buttons
- **Gallery** - Product categories showcase (Birthday Cakes, Themed Cakes, Cupcakes)
- **Our Artistry** - Premium ingredients and craftsmanship highlights
- **Reviews** - Auto-rotating customer testimonials
- **Booking Form** - Contact form for cake enquiries
- **Footer** - Business information and contact details

🔧 **Responsive Design**
- Mobile-optimized layouts
- Flexible grid system
- Touch-friendly buttons

📱 **Interactive Elements**
- Form validation
- Auto-rotating review carousel (5-second intervals)
- Smooth hover effects
- Scroll animations

## How to Use

1. Clone this repository or download the files
2. Open `index.html` in your web browser
3. The website is fully functional and ready to use!

## Customization

### Update Business Information
Edit these sections in `index.html`:
- Phone number: Replace `07547554323`
- Email: Replace `orders@bakerydelights.co.uk`
- Address: Update the footer address
- Business name: Modify "JAMILA'S" in the logo

### Modify Cake Styles
Update the style options in the booking form:
```html
<option value="your-style">Your Cake Style Description</option>
```

### Change Colors
Edit the CSS variables at the top of the `<style>` section:
```css
:root {
    --cream: #FDFBF7;
    --soft-pink: #FFF0F2;
    --dark-berry: #2C1A1D;
    --gold: #D4AF37;
    --muted-gold: #AA7C11;
    --white: #FFFFFF;
}
```
